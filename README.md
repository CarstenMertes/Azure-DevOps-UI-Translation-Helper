# Azure-DevOps-UI-Translation-Helper

[![Edge](https://img.shields.io/badge/Edge-Compatible-green)](https://microsoftedge.microsoft.com/addons/) [![Chrome](https://img.shields.io/badge/Chrome-Compatible-blue)](https://chrome.google.com/webstore/category/extensions) [![MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Translate and customize the Azure DevOps user interface to your preferred language.

## Features
- Automatic translation of UI elements
- Easy language switching
- Support for multiple languages
- Customizable translation rules

## Installation
1. Download the extension as a .crx file or clone this repository.
2. Open Microsoft Edge and go to `edge://extensions/`.
3. Enable Developer mode.
4. Click "Load unpacked" and select the extracted folder.

**For Chrome:** Use `chrome://extensions/` with the same steps.
